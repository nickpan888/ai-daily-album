01 AI 开始管你的钱了

今天 OpenAI 干了一件事，听起来不大，但仔细想想，可能是今年最重要的一次产品更新。

ChatGPT 现在能连接你的银行账户了。通过 Plaid 接入，用户可以绑定银行账户进行预算管理和消费分析——每一笔收入和支出，AI 都能看到。

这跟查余额那种肤浅功能完全不是一个物种。这是 OpenAI 在跨越一条红线——从"信息层"进入"交易层"。处理信息，错了最多丢人。处理资金，错了丢的是真金白银。

从对话 → 交易 AI 能力边界的历史性跨越

而且有意思的是，OpenAI 没自建银行接口，而是用了 Plaid——就是那个连接银行和各类金融应用的中间层。这说明策略很清楚：先占位，再优化。快速上线、快速获取用户、快速积累金融数据。

问题是：你准备好把银行密码告诉 AI 了吗？OpenAI 要攻克的不是技术难题，是信任难题。而信任，恰恰是金融行业的根基。

02 微软收网：Claude Code 出局

第二件大事，发生在开发者工具领域。

微软正在逐步淘汰 Claude Code，引导内部开发者全面转向 GitHub Copilot CLI。这不是一个普通的"换供应商"决定，背后是微软在 PC 时代最擅长的策略——捆绑。

"历史不会重复，但总在押韵。微软的 AI 时代捆绑策略，和它的 PC 时代用的是同一本剧本。"

当年 IE 捆绑 Windows 打垮了 Netscape，Teams 捆绑 Office 365 击退了 Slack。现在，GitHub Copilot 捆绑 Azure + GitHub 生态，目标是谁？不言自明。

对开发者来说，好消息是 Copilot 和 GitHub 的整合确实体验更好；坏消息是——你又一次被锁在了一个生态里。

03 OpenAI 的"成人礼"

第三件事，发生在 OpenAI 内部。

Greg Brockman 重新出山担任产品战略负责人。ChatGPT 和 Codex 合并。公司为 IPO 做最后冲刺。三个信号指向同一个结论：OpenAI 正在从"研究实验室"变成一家"正常"的科技公司。

把对话和编程合并的逻辑很有意思。OpenAI 认为，未来用户不需要区分"聊天"和"编程"——AI 会自动理解需求并切换模式。这就是 AI Agent 的雏形。

IPO 更加直白。上市意味着 OpenAI 需要展示可预测的财务表现，会影响它敢冒的风险。研究实验室随心所欲探索的时代，正式结束了。
