# AI 热点画册 · 每日狩猎报告

每天一期，用深度视角看懂 AI 圈正在发生的本质变化。

## 访问地址

**最新一期：** https://nickpan888.github.io/ai-daily-album/

**历史存档：** https://nickpan888.github.io/ai-daily-album/archive

每期独立链接：`/YYYYMMDD`（如 `/20260514`）

## 内容结构

每期画册包含以下板块：

- **头条深度** — 当日最重要的 AI 事件，配底层逻辑 + 行业变量 + 科技侠点评
- **垂直战场** — 法律、医疗、教育等垂直领域的 AI 渗透
- **平台战争** — AI 代理、搜索、硬件入口的平台级重构
- **资本 · 供应链 · 基础设施** — 资本流向、能源争议、地缘博弈
- **安全 · 治理 · 劳动力** — 监管框架、数据安全、就业冲击
- **关键数据一览** — 核心指标速览
- **主编结语** — 每日主题提炼

## 数据源

- [Brief.ai](https://www.brief.news/ai)

## 更新频率

每日更新，自动同步至 GitHub Pages。

## 本地部署

```bash
git clone https://github.com/nickpan888/ai-daily-album.git
cd ai-daily-album
# 直接用浏览器打开 index.html 即可预览
```
